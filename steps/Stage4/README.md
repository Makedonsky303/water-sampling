# Stage 4: Транспортировка и сдача в лабораторию

Симулятор этапа транспортировки проб воды и приёмки в лаборатории.

## 📁 Структура файлов

```
steps/Stage4/
├── index.jsx              # Главный компонент с навигацией и отчётом
├── Step1_PackBag.jsx      # Шаг 1: Укладка термосумки (drag-and-drop)
├── Step2_SealBag.jsx      # Шаг 2: Герметизация (интерактивная молния)
├── Step3_Transport.jsx    # Шаг 3: Транспортировка (3 сценария)
└── Step4_LabReception.jsx # Шаг 4: Приёмка в лаборатории
```

## 🎯 Особенности

### ✅ Требования из чата выполнены:

1. **Без явных подсказок** - система не говорит "правильно/неправильно" во время прохождения
2. **Drag-and-drop укладка** - студент сам размещает предметы в термосумке
3. **Визуальная оценка** - студент видит температуру, состояние флаконов и сам принимает решения
4. **Иммерсивная герметизация** - интерактивная молния (drag бегунка) и фиксаторы
5. **Динамика маршрута** - 3 сценария (1.5ч / 4ч / 6.5ч) с разными условиями
6. **Отчёт только в конце** - детальная оценка всех шагов показывается в финале

### 📋 Шаги симулятора:

#### Шаг 1: Укладка термосумки
- Drag-and-drop система размещения предметов
- 4 зоны: левая стенка, правая стенка, центр (перед), центр (зад)
- 8 предметов (включая 3 дистрактора)
- Правильная укладка: хладоэлементы по бокам, перегородка, флаконы

#### Шаг 2: Герметизация
- Интерактивная молния (drag бегунка по пути)
- 2 фиксатора-защёлки (клик для закрытия)
- SVG-анимация в реальном времени

#### Шаг 3: Транспортировка
- Выбор из 3 сценариев:
  - **Короткий** (1.5ч): без проблем, доклад не нужен
  - **Средний** (4ч): небольшой рост температуры
  - **Длинный** (6.5ч): затор, критическая температура, требуется доклад
- Реалистичная температурная кривая (tempCurve)
- Журнал событий с алертами
- Возможность доложить в лабораторию (для >5ч)

#### Шаг 4: Приёмка в лаборатории
- Визуальный осмотр: студент видит реальное состояние
- Заполнение акта приёмки на основе наблюдений
- Проверка: температура, флаконы, герметичность, этикетки

### 📊 Финальный отчёт
- Общий балл из 100
- Детализация ошибок по каждому шагу
- Оценка качества (отлично/хорошо/удовлетворительно)
- Ссылки на ГОСТы и СП

## 🚀 Использование

### Импорт в Next.js:

```jsx
import Stage4Simulator from '@/steps/Stage4';

export default function Stage4Page() {
  const handleComplete = (results) => {
    console.log('Результаты этапа 4:', results);
    // Здесь можно отправить результаты на сервер или перейти к следующему этапу
  };

  return <Stage4Simulator onComplete={handleComplete} />;
}
```

### Callback `onComplete`:

Возвращает объект:

```javascript
{
  step1: {
    packingErrors: ['Описание ошибки 1', ...],
    packingScorePenalty: 15,
    packingData: { leftWall: 'ice1', ... }
  },
  step2: {
    sealingErrors: [...],
    sealingScorePenalty: 20,
    sealingData: { zipperProgress: 0.98, ... }
  },
  step3: {
    transportErrors: [...],
    transportScorePenalty: 25,
    transportData: { scenario: 'long', finalTemp: 9.8, ... }
  },
  step4: {
    receptionErrors: [...],
    receptionScorePenalty: 10,
    receptionData: { tempReading: 9.8, ... }
  }
}
```

## 📖 Нормативная база

- **ГОСТ Р 59024‑2020** — Отбор проб питьевой воды
- **СП 1.3.3118-13** — Безопасность работы с микроорганизмами
- **МУК 4.2.1018-01** — Микробиологический мониторинг
